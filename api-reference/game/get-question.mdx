---
title: 'Get Question'
api: 'GET /api/game/question'
description: 'Generate a new crafting question for the game'
---

## Query Parameters

<ParamField query="difficulty" type="string" default="MEDIUM">
  The difficulty level of the question. Valid values are:
  - `EASY` - Simple items with fewer components
  - `MEDIUM` - Moderate complexity items
  - `HARD` - Complex items with multiple components
  
  This parameter is optional and defaults to `MEDIUM` if not provided.
</ParamField>

## Response

<ResponseField name="targetItemId" type="string" required>
  The unique identifier of the target item to craft
</ResponseField>

<ResponseField name="targetItemName" type="string" required>
  The display name of the target item
</ResponseField>

<ResponseField name="targetItemImageUrl" type="string" required>
  URL to the image of the target item
</ResponseField>

<ResponseField name="correctComponentIds" type="List<String>" required>
  List of item IDs that are the correct components to craft the target item
</ResponseField>

<ResponseField name="correctComponents" type="List<ItemOption>" required>
  List of ItemOption objects representing the correct components with full details
  
  <Expandable title="ItemOption properties">
    <ResponseField name="itemId" type="string">
      The unique identifier of the component item
    </ResponseField>
    
    <ResponseField name="name" type="string">
      The display name of the component
    </ResponseField>
    
    <ResponseField name="imageUrl" type="string">
      URL to the component's image
    </ResponseField>
    
    <ResponseField name="cost" type="integer">
      The gold cost of the component
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="options" type="List<ItemOption>" required>
  List of all available options including correct components and distractors. Players must select the correct components from this list.
</ResponseField>

<ResponseField name="timeLimit" type="integer" required>
  Time limit in seconds for answering this question
</ResponseField>

<ResponseField name="difficulty" type="string" required>
  The difficulty level of this question (EASY, MEDIUM, or HARD)
</ResponseField>

## Example Request

```bash
curl -X GET "https://api.crafterlol.com/api/game/question?difficulty=HARD"
```
